naviertwin.core.flow_analysis
============================================================

Customer-facing CFD post-processing helpers for turning raw fields into
statistics, spectra, forces, probes, reduced modal diagnostics, and
thermofluid quantities. The package root exposes the common post-processing
operations lazily, so importing ``naviertwin.core.flow_analysis`` does not pull
in optional modal-analysis backends.
