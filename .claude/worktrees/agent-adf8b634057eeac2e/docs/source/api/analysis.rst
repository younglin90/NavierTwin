naviertwin.core.analysis
============================================================

General CFD analysis utilities for customer workflows that sit between raw
post-processing and model training. The package root exposes lightweight latent
embedding helpers, while the submodules provide focused analysis routines for
boundary layers, spectra, vortex structures, and spatial probes.
